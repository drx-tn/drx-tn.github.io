MPTabs est une application qui permet de consulter rapidement les tableaux des maladies professionnelles en Tunisie. Elle s'inspire et utilise des documents publiés par l'Institut de Santé et de Sécurité au Travail.

C'est une application c++ utilisant le framework Qt pour les version Windows, Linux et macOs X et une application Kotlin pour sa version Android.

Le code source est disponible sur le dépôt GitHub à cette adresse. Vous pouvez rapporter les erreurs, bugs et autres sur cette page. Votre contribution est souhaitée.

MPTabs est disponible en Téléchargement dès à présent à cette page. Une version en ligne est consultable aussi à cette adresse.
