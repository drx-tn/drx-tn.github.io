QRCheck

Vérification des Cachets Electroniques TN CEV 2D-Doc

A propos :

QRCheck est une application qui a été développée par moi, Dr. Nizar Ben Ltaief, médecin à plein temps au Groupement de Médecine du Travail du Gouvernorat de Sousse.

Cette application correspond en fait à un refactoring de l'application QR-Check développée pour la plateforme Android par l'Agence Nationale de Certification Electronique TUNTRUST.

Contrairement à cette dernière, QRCheck est une application Progressive Web App pouvant être utilisée en ligne ou installée et utilisée en hors ligne sur les téléphones et tablettes Android et iOs ainsi que sur les ordinateurs PC Windows, Linux et Mac OS. QRCheck permet aussi de basculer entre les différentes caméras pouvant être connectées au système utilisé.

Le but de cette application est de faciliter la vérification de la validité des passes vaccinaux et des certificats de vaccination.

Accéder à QRCheck

Cette application n'est qu'un refactoring de l'application QR-Check développée pour la plateforme Android par l'Agence Nationale de Certification Electronique TUNTRUST : je ne détiens aucun droit sur son code source; les droits originaux des différentes librairies et codes utilisés reviennent à leurs auteurs originaux.

Installation :

QRCheck est une application Progressive Web App (PWA) pouvant être installée depuis plusieurs navigateurs internet massivement utilisés tels que Chrome, Firefox, Opera ou encore Safari que ce soit sous PC, Mac, iPhone, iPad ou encore téléphones et tablettes Android.

Sous bien nombre de navigateurs vous pouvez être invités à installer cette application.

Sous Chrome sous Android par exemple vous pouvez installer l'application depuis l'invite à ajouter l'application à votre accueil ou bien à partir du menu de Chrome.

Sous Chrome pour PC vous pouvez cliquer sur l'icone qui se trouve à côté de la barre d'adresse du navigateur.

Vous trouverez ensuite l'application QRCheck sur votre bureau.

Retrouvez le code source de cette application sur ce dépôt GitHub : https://github.com/drx-tn/qrcheck.
