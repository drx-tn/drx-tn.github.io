MPTabs

Application tableaux des maladies professionnelles en Tunisie

A propos :

MPTabs est une application qui permet de consulter rapidement les tableaux des maladies professionnelles en Tunisie. Elle s'inspire et utilise des documents publiés par l'Institut de Santé et de Sécurité au Travail.

MPTabs existe en versions multiplateformes avec des versions pour Windows, Mac Os, Linux et Android avec aussi une version consultable en ligne.

C'est une application gratuite et son code source est ouvert.

Retrouvez le code source de l'application MPTabs ainsi que les instructions pour la compiler à partir de la source sur ce dépôt GitHub.

Téléchargement :

macOS X

ou Télécharger APK (déconseillé)

Pour installer MPTabs sous Ubuntu (Bionic / Cosmic / Disco / Eoan) ajoutez le PPA ppa:drx.tn/ppa à vos sources de mises-à-jour puis installez le paquet mptabs avec ces commandes:


    sudo add-apt-repository ppa:drx.tn/ppa
    sudo apt-get update
    sudo apt-get install mptabs

Pour les autres versions de Linux, vous devez compiler par vous-même l'application à partir de la source.
